Flask Stores Rest Api

The Rest Api built with Flask Restful , Flask-JWT , Flask-SQL Alchemy and Portgree SQL and these are deployed on HEROKU. The Api contains User Registration , Authentication and a database to store items and stores. The rest Api are flexible and easy to integrate with your application and also easy to modify as per your frontend applicaion and requirement To rest the Api , we strongly recommend to test with :
Postman development environment.

End points of API

API methods:

GET ---------------------------------> '/' Home

POST --------------------------------> '/register' - User Registration

POST ---------------------------------> '/auth - User authentication

GET ----------------------------------> '/items' - To get all the items

GET , POST ------------------------> '/item/< string:name>' - To get and post the item

GET ----------------------------------> '/stores' - To get all the stores

GET , POST ------------------------> '/store/< string:name>' - To get and post the store
